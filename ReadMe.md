# Freshlane Website

*Author:* Leon  
*Description:* Freshlane youth fashion website showcasing products, team, and enquiry forms.  
*Structure:* HTML, CSS, JS (Vanilla)  
*CSS:* Upgraded modern style with fade-in effects, responsive design.

## Pages
- index.html → Home / Hero / Brand Story
- pages/products.html → Product Gallery
- pages/team.html → Founder & Team
- pages/enquiry.html → Enquiry Form
- pages/contact.html → Contact Info & Map

All images are in /images/ folder with exact names as referenced in HTML.

- Responsive navigation with hamburger menu
- Hero section with main banner
- Product gallery
- Team / Founder sections
- Enquiry & Contact forms
- Fade-in scroll animations
