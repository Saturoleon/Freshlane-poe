*Description:*  
Freshlane is a youth-focused fashion website that showcases our clothing brand’s products, team, and brand story. The website is designed to be modern, responsive, and easy to navigate, with a focus on enhancing user experience and engagement.  

*Technologies Used:*  
- HTML5  
- CSS3 (Modern & responsive styling)  
- JavaScript (Vanilla)  
- Responsive design with fade-in scroll animations  

*Folder Structure:*

freshlane/ │ ├── index.html           # Home / Hero / Brand Story ├── pages/ │   ├── products.html    # Product Gallery │   ├── team.html        # Founder & Team │   ├── enquiry.html     # Enquiry Form │   └── contact.html     # Contact & Map ├── css/ │   └── style.css        # Modern CSS styling ├── js/ │   └── main.js          # JS for navigation & animations ├── images/              # All image assets │   ├── Freshlane Logo.png │   ├── Mission Section.png │   ├── Leon.png │   ├── Meet Our Team.png │   ├── Hoodie.png │   ├── Outfit 2.png │   ├── Outfit 3.png │   └── Our Story.png ├── README.md └── CHANGELOG.md

*Features:*
- Sticky navigation bar with hamburger menu for mobile
- Hero / Main banner showcasing mission
- Brand story section with image
- Product gallery with hover effects
- Founder and team section
- Enquiry form for customers
- Contact page with embedded map
- Responsive layout across devices
- Smooth fade-in animations on scroll

*Usage:*  
1. Open index.html in a browser to view the homepage.  
2. Navigate through the site using the navigation bar or hamburger menu.  
3. All pages are linked properly and assets are stored in the images/ folder.  
4. Forms (Enquiry) are front-end ready; backend integration can be added if required.
