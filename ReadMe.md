# Freshlane | Walk Your Lane

## Project Description

Freshlane is a youth-focused streetwear brand website showcasing hoodies and cargo pants. The website is fully responsive, visually appealing, and provides easy navigation for users to view products, learn about the brand, and contact the team.

The website includes interactive elements, dynamic real-time content, and a gallery lightbox for products, ensuring an engaging user experience.

## How to Run Locally

1. Clone the repository or download the ZIP file.
2. Open `index.html` in a web browser.
3. Navigate using the menu or the **hamburger menu** on smaller screens.
4. Test interactive elements such as:

   * Product image previews
   * Live date and time in the footer
   * Hamburger menu toggle
   * Forms and Google Map section



## Website Features

* Fully responsive layout (desktop, tablet, mobile)
* Hamburger menu for smaller screens
* Featured products with hover effects and **image preview modal**
* About Us page with team introduction and **Google Map iframe**
* Enquiry and Contact forms with **JavaScript input validation & feedback**
* Live date & time display on all pages
* Interactive JavaScript elements including modals, lightboxes, and accordion example
* Consistent design, typography, and colors
* Buttons and interactive elements with hover effects
